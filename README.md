# 🎉 Event Management System

A responsive **Event Management System** built using **HTML**, **CSS**, and **JavaScript**. The application allows users to browse events, register for events, navigate through dashboards, and submit contact information through a clean and interactive web interface.

## Project Structure

```text
EventManagementSystem/
├── index.html
├── Dashboard.html
├── Events.html
├── Register.html
├── Contact.html
├── css/
├── JS/
└── images/
```

## Features

- Browse available events
- Event registration form
- Interactive dashboard
- Contact page
- Responsive UI
- Client-side form validation
- Organized assets (CSS, JavaScript, Images)

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Structure |
| CSS3 | Styling |
| JavaScript | Client-side functionality |

## Getting Started

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Navigate through the application using the menu.
